# Playlist-Pro
A playlist management web app where users can create and organize playlists with features like sorting and mood tagging.

## Features
- Playlist Management (Create, rename, and delete playlists, mark playlists as favorites, and view all playlists in a clean sidebar layout)
- Add songs with title and artist
- Edit and delete songs
- Prevent duplicate songs (same title + artist)
- Sort songs by: Artist (A–Z), Title (A–Z), Recently added
- Filter playlists by mood (Add tags like happy or sad)
- Mark/unmark playlists as favorites
- Filter to show only favorite playlists
